Research Papers

JSModel: Analyzing and Categorizing JavaScript in Today's Web Using Machine Learning

In this paper, we present JSModel, a novel machine learning based approach to classifiy JS code present in the web-pages with a high re- call and precision. We analyze JavaScript elements from a real-world data-set and classifying them into 15 different JS categories. The fine-grained analysis results along with our detailed methodology of building the vocabulary and vectors for the Machine Learning models are presented in our work. JSModel classification can be used to simplify complex web pages and decrease overall page load time and improve the websitesâ€™ performance. The paper is submitted in ACM Sigmetrics 2020 and I am the primary author of this research paper.

Using Application Layer Banner Data to Automatically Identify IoT Devices

This is an accepted Sigcomm CCR 2020's paper. In this paper, we claim that a work from Usenix on automatic identification of IoT devices is not an efficient procedure and resuts in ~19% accuracy on real world data-set which is significantly lower than the reported in Usenix's paper. I am a second author in this paper and I started this work upon securing a grant from Lahore University of Management Sciences (LUMS). Previously, a poster related to this work was also accepted in IMC Reproducibility Track 2019.

Half-Hulk Chrome Browser Extension | JS | Chrome API

I have made an extension for chrome browser which insures that other installed browser extensions are not causing any security or privacy related issues, like user's data stealing, while performing their network activities. It watches actions of other extensions in asynchronous mode and triggers a scan when any suspecious activity seems to occur. I developed it after motivation from a research paper published by UC Berkeley.
